# Lotto Number Generator

## Overview

A web application that generates lottery numbers based on historical winning number analysis. It provides two modes: one for generating numbers with a higher probability of winning based on analysis, and another for generating numbers based on frequency.

## Project Outline

### Style and Design
- Modern, clean, and intuitive user interface.
- Responsive design for both mobile and web.
- Use of modern CSS features for a visually appealing layout.
- Clear typography and iconography to enhance user experience.

### Features
- **Lotto Number Generation:**
  - Generate 6 random lottery numbers.
  - "Get Lotto Numbers" button to trigger the generation.
- **Frequently Appearing Numbers:**
  - "Frequently Appearing Numbers" button to generate numbers based on historical frequency.
- **Display Results:**
  - Display the generated numbers clearly on the screen.

## Implemented Features

- Initial HTML structure with buttons and display area.
- Modern and visually appealing styles using CSS.
- JavaScript logic for generating and displaying lottery numbers.

## Current Plan

- Modify the number of generated lottery numbers from 5 to 6 to align with the standard Korean lotto format.
